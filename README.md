# excel-lookup-functions
Solved real business lookup questions using VLOOKUP and XLOOKUP to retrieve customer information, sales values, customer segments, and regional data from a structured sales dataset.
## Project Overview

This project demonstrates how to use both VLOOKUP and XLOOKUP to answer real business questions using Microsoft Excel.

## Business Questions

- What Customer Segment does Mohammed MacIntyre belong to?
- What was the sales amount on 5/7/2009?
- Who generated sales of 4158.1235?
- Which customer belongs to the Yukon region?

## Excel Functions Used

- VLOOKUP
- XLOOKUP

## Skills Demonstrated

- Data Retrieval
- Lookup Functions
- Business Analysis
- Exact Match Searches
- Structured Excel Tables
